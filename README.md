# Electric-Vehicle-Range-Prediction-Using-Machine-Learning
This project predicts the driving range of an Electric Vehicle using machine learning models based on battery capacity, speed, and other vehicle parameters.
## Technologies Used
- Python
- Pandas
- Scikit-learn

## Dataset Features
- Battery Capacity (kWh)
- Vehicle Weight (kg)
- Average Speed (km/h)
- Temperature (°C)

## Project Files
- EV_range_prediction.py : Machine learning model code
- ev_dataset.csv : Dataset used for training and testing
- requirements.txt : Required Python libraries

## How to Run the Project

Install the required libraries:

pip install -r requirements.txt

Run the program:

python EV_range_prediction.py

## Output
The model predicts the estimated driving range of an electric vehicle based on given input parameters.
